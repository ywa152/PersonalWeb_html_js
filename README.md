# 不叫奥神.cn 开源网站项目说明文档

## 教会你从零开始搭建个人网站

## 项目概述
本项目是一个集软件推广下载与多媒体内容分享为一体的综合性网站（域名：不叫奥神.cn）。网站核心分为两大模块：主页面作为软件推广平台，提供软件安装包的在线展示与下载服务；副页面具备类似B站、抖音的内容分享功能，支持注册用户上传视频、图片类内容，并向其他用户开放在线浏览权限。

## 技术栈
### 前端技术
- HTML：构建网站页面的基础结构
- CSS：实现页面样式美化与布局控制
- JavaScript：处理页面交互逻辑、表单验证、内容上传/展示等动态功能

### 后端/数据存储
- MySQL：用于存储用户信息、软件信息、多媒体内容信息等核心数据

### 第三方服务
- Email-JS：实现用户注册时，向用户填写的QQ邮箱发送注册通知/验证消息

## 功能说明
### 1. 核心功能模块
| 模块       | 功能描述                                                                 |
|------------|--------------------------------------------------------------------------|
| 软件推广下载 | 1. 展示软件基本信息（名称、简介、版本、适用系统等）<br>2. 提供软件安装包的在线下载功能<br>3. 支持软件分类/搜索（可选扩展） |
| 多媒体内容分享 | 1. 用户注册/登录功能<br>2. 注册用户可上传视频、图片类内容<br>3. 所有用户可在线浏览已发布的视频、图片内容<br>4. 内容展示/分页/播放等基础交互 |

### 2. 辅助功能
- 用户注册：填写信息后，通过Email-JS向QQ邮箱发送注册成功/验证消息
- 基础页面导航：主页面与副页面的切换跳转

## 环境部署要求
### 基础环境
- 操作系统：Windows/Linux/macOS 均可
- Web服务器：Apache/Nginx/IIS（需支持PHP/Node.js等后端语言，根据实际开发补充）
- 数据库：MySQL 5.7+ / 8.0+
- 浏览器：Chrome/Firefox/Edge 等现代浏览器（兼容ES6+）

### 第三方依赖配置
1. 注册Email-JS账号，获取 `user ID`、`service ID`、`template ID`
2. 在项目的JS代码中配置上述ID，确保与QQ邮箱发送逻辑对接正常
3. MySQL数据库需创建对应表（用户表、软件表、内容表等），示例表结构可参考 `sql/init.sql`（可选）

## 快速启动
1. 将项目文件放入Web服务器的根目录（如Apache的 `htdocs` 目录）
2. 配置MySQL数据库连接信息（修改项目中数据库配置文件）
3. 配置Email-JS相关参数（修改前端JS中的邮箱发送配置）
4. 启动Web服务器和MySQL服务
5. 访问网站根目录（如 `http://localhost/`）即可进入主页面

## 注意事项
1. 确保MySQL服务正常运行，且数据库连接账号具备足够的读写权限
2. Email-JS的API密钥/ID需妥善保管，避免泄露
3. 软件安装包和用户上传的多媒体文件需做好存储管理，定期清理无效文件
4. 若涉及大文件上传（视频），需调整Web服务器的上传大小限制

## 总结
1. 本网站核心为“软件推广下载+多媒体内容分享”双模块，技术栈以HTML/CSS/JS为前端基础，MySQL为数据存储，Email-JS实现注册邮箱通知；
2. 部署时需重点配置数据库连接和Email-JS第三方服务，确保核心功能正常运行；
3. 目录结构采用模块化划分，便于后续功能扩展和维护。
